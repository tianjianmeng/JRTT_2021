## 本地开发

```sh
# 下载源码
git clone https://github.com/lipengzhou/topline-m.git

# 安装依赖
# 或者 npm install
yarn install

# 启动开发服务
# 或者 npm run serve
yarn serve

# 打包构建
# 或者 npm run build
yarn build

# 代码格式校验
# 或者 npm run lint
yarn lint
```
